CSS Flexbox

Flexbox is a layout model that is based in one dimension instead of two(like with Grid Layout). This means when placing an object you have to specifiy one dimension at a time.

example

In this example we are creating a series of horizontaly placed boxes that will be rearranged when the page window is resized

note that the classes flex container and flex item are simply to tie the css to these element. The names of the classes dont effect anything as the functionality comes from the CSS

Most of the css here is just to make the boxes look nice. The properties that effects the flexbox are:

  • display: flex; this shows that the display mode is flex
  • flex-flow: row wrap; this set the direction of the flex to row (placing each element in a row) and tells the container to wrap (keeps the elements inside the container and in the correct order)
  • justify-content: space-around; this tells the container to add space around each element inside it

Further Notes

The other options for flex-flow directions are : collumn where the elements are orginised virtically

The other options for flex-flow wrap are:

  • no-wrap - this means that the elements are all forced on one line
  • wrap-reverse - this means that the order of the lines the elements are placed on is reversed
